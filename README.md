## Prerequisites

- npm >=6.14.17
- node >=14.19.3
- python >=3.6.0

## Install

```sh
git clone <repo-url>

pip install -r requirements.txt
```

## Usage

```sh
python server.py
```

## Server Deployment

```sh
if __name__ == '__main__':
    app.run(host='<ip-address>', port=<port>)
```

## Angular Installation

```sh
npm install
```

## Angular Usage

```sh
ng serve
```
